# 身心灵疗愈日报 (Healtherly Daily Report)

一个专注于身心灵健康的每日疗愈内容展示平台，旨在通过冥想引导、心理调适、正念练习等方式，帮助用户开启充满正能量的一天。

## 特性

- 🌿 **极简设计** - 采用暗色主题，营造宁静的氛围
- 🧘 **冥想引导** - 提供呼吸练习和冥想指导
- 🧠 **心理调适** - 每日心理调适小贴士
- 🌸 **正念练习** - 日常正念觉察练习
- 🙏 **感恩日记** - 每日感恩事项记录
- 📱 **响应式设计** - 支持各种设备访问

## 设计灵感

该项目的设计灵感来源于优秀的产品沉思录网站 (https://read.pmthinking.com)，采用了类似的极简暗色主题和优雅的交互动效。

## 功能特色

1. **每日疗愈语录** - 精选正面积极的心灵寄语
2. **冥想引导** - 提供呼吸练习和冥想指导，包含交互式呼吸动画
3. **心理调适** - 情绪管理和心理调节技巧
4. **正念练习** - 日常正念觉察练习
5. **感恩日记** - 每日感恩事项记录
6. **往期回顾** - 查看历史疗愈内容

## 使用方法

### 方法一：使用简易服务器（推荐）
```bash
cd healtherly-daily-report
node server.js
```
然后在浏览器中访问 `http://localhost:3000`

### 方法二：直接打开文件
直接打开 `public/index.html` 文件即可在浏览器中预览

### 方法三：使用启动脚本
```bash
cd healtherly-daily-report
bash start.sh
```

## 部署到 GitHub

要将此项目部署到您的 GitHub 账户，请使用以下方法之一：

### 方法一：使用部署脚本
```bash
cd healtherly-daily-report
bash deploy_to_github.sh
```

### 方法二：手动配置
请参阅 `GITHUB_SETUP.md` 文件中的详细说明。

## 开发理念

本项目遵循以下设计原则：

1. **简约至上** - 界面简洁，突出内容本身
2. **情感化设计** - 使用柔和色彩和舒适间距
3. **功能性优先** - 每个元素都有其存在的意义
4. **流畅体验** - 优化动画和过渡效果
5. **包容性** - 考虑不同用户的需求和能力
